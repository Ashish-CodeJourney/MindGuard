Why Android warns you

MindGuard isn't on the Google Play Store, so you'll see a Play Protect warning during install. That's normal for sideloaded apps — Google simply can't vet it through their store, not a sign it's unsafe. Tap "Install anyway" to proceed. The full source is on GitHub if you want to check for yourself.

Get started in two minutes

1Download the latest APK from the GitHub release, or use the button up top.
2Android will prompt you — tap Settings → Allow from this source, then install.
3Go to Settings → Apps → MindGuard → ⋮ → Allow restricted settings.
Android 13+
4Go to Settings → Accessibility → MindGuard and turn it on.
✓Done. Open Instagram or YouTube — Reels and Shorts stop cold.
