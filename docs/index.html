About Play Protect Warnings

Since MindGuard isn't on Google Play Store, you'll see a Play Protect warning during installation. This is normal for sideloaded apps—Google can't verify it through their store, but that doesn't mean it's unsafe. Tap "Install anyway" to proceed. The code is open source on GitHub for full transparency.

1

Download the APK

3

Go to Settings → Accessibility → MindGuard and toggle it on.
