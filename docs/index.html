How it works

Simple, transparent, private

MindGuard uses Android's Accessibility Service to watch for short-video players — and only those.

1

Accessibility Service

MindGuard reads view IDs within the 4 target apps only — no other apps are ever observed.

2

Player Detection

Only player-specific resource IDs trigger a block. Opening YouTube or Instagram normally is never interrupted.

3

Instant Navigation

The moment a player is detected, MindGuard navigates away — to your Home feed on Instagram, or back on other apps.

4

Cooldown Guard

A 500ms cooldown prevents rapid back-press loops so the block fires cleanly every time.

Installation

Up and running in 2 minutes

MindGuard is a sideloaded APK — no Play Store required.

1

Download the APK

Tap the button above or download directly and open the APK on your device.

2

Allow installation from unknown sources

Android will prompt you — tap Settings → Allow from this source, then install.

3

Allow restricted settings Android 13+ only

Go to Settings → Apps → MindGuard → ⋮ → Allow restricted settings.

4

Enable the accessibility service

Go to Settings → Accessibility → MindGuard and flip the toggle.

5

Done

MindGuard is now active. Open Instagram or YouTube — Reels and Shorts are blocked.
